# 使用指南 (User Guides)

歡迎來到 Taiwan Health MCP 使用指南。本區塊提供以任務為導向的教學，協助您將不同的工具組合起來，解決實際的醫療或健康問題。

## 內容單元

### [完整診療流程模擬](complete-workflow.md)
演示如何從病患主訴開始，串聯 ICD 查詢、藥品選擇、FHIR 記錄生成的完整流程。適合開發者了解各模組間的協作方式。

### [藥品查詢與識別](drug-identification.md)
專注於藥物相關應用，包括如何搜尋藥品詳情、辨識不明藥丸以及進行簡單的用藥安全檢查。

### [檢驗結果判讀](lab-interpretation.md)
指引如何使用本系統來解讀生化檢驗報告，並了解異常數值的臨床意義。

### [LOINC 整合指南](loinc-integration.md)
針對醫療資訊人員 (IT)，說明如何利用本系統將院內代碼標準化為 LOINC 編碼。

### [臨床路徑規劃](clinical-pathway.md)
展示如何結合臨床指引工具，為特定疾病規劃標準化的照護路徑。
