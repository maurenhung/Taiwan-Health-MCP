# LOINC 資料來源

## 國際標準
- **來源權威**：Regenstrief Institute
- **版本**：目前系統支援 LOINC 2.76 版本。
- **內容**：包含超過 70,000 筆實驗室檢驗與臨床觀察代碼。

## 台灣中文化對應
- **來源**：衛福部電子病歷推動專區 - 台灣臨床檢驗代碼對應表。
- **加值處理**：
    - 將官方英文名稱與台灣常用臨床術語（如「飯前血糖」對應 "Glucose pre meal"）進行關鍵字索引。
    - 建立檢體類別 (Specimen) 與檢驗方法 (Method) 的中文別名庫。

## 參考值資料
- **來源**：彙整自國內數家醫學中心之公開檢驗手冊（如台大醫院、榮民總醫院）。
- **注意**：各實驗室因儀器試劑不同，參考值可能略有差異，本系統提供之參考值僅供通用性參考。
