# 臨床指引資料來源

本系統的臨床指引資料庫是由專案團隊人工整理並結構化後的成果，原始資料來自以下公開管道：

## 國內來源
1. **衛生福利部**：國民健康署發布之慢性病防治指引。
2. **專科醫學會**：
    - 中華民國糖尿病學會
    - 台灣高血壓學會
    - 台灣心臟學會
    - 台灣胸腔暨重症加護醫學會

## 國際來源
為了補足國內指引未覆蓋的領域，部分參考：
- **美國**：AHA (心臟), ADA (糖尿病), NCCN (癌症)。
- **歐洲**：ESC (心臟)。

## 資料結構化
原始指引多為 PDF 文件，本專案將其轉換為 JSON 結構，包含：
- 流程圖 (Flowchart) 邏輯
- 用藥表格 (Medication Table)
- 評估量表 (Scales)

此結構化過程確保了 MCP 工具能精確回傳機讀 (Machine-readable) 的建議內容。
