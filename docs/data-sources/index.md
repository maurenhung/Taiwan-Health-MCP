# 資料來源概覽

本系統所使用的醫療數據均來自權威的公開資料庫，經過整理、清洗與標準化後存入本地資料庫。

## 主要來源列表

| 資料類別 | 主要來源 | 更新頻率 |
| :--- | :--- | :--- |
| **藥品** | 台灣衛生福利部食品藥物管理署 (TFDA) | 每月 |
| **健康食品** | 台灣衛生福利部食品藥物管理署 (TFDA) | 每季 |
| **食品營養** | 台灣食品成分資料庫 | 不定期 |
| **ICD-10** | 美國 CMS / 台灣健保署 | 每年 |
| **LOINC** | Regenstrief Institute / 台灣電子病歷辦公室 | 每半年 |
| **臨床指引** | 各大醫學會 / 衛福部 | 不定期 |
