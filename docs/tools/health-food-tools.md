# 健康食品工具 (Health Food Tools)

此類別工具依據台灣《健康食品管理法》規範，提供核可健康食品的查詢與應用分析。

## search_health_food
搜尋健康食品。

### 參數
| 參數名 | 型別 | 必填 | 說明 | 範例 |
| :--- | :--- | :--- | :--- | :--- |
| `keyword` | string | 是 | 產品名稱或保健功效 | `"靈芝"`, `"調節血脂"`, `"護肝"` |

---

## get_health_food_details
取得健康食品的詳細核定資料。

### 參數
| 參數名 | 型別 | 必填 | 說明 | 範例 |
| :--- | :--- | :--- | :--- | :--- |
| `license_number` | string | 是 | 衛部健食字號 | `"衛部健食字第A00123號"` |

### 回傳內容
- 核准之保健功效
- 功效成分含量
- 警語與攝取注意事項

---

## analyze_health_support_for_condition
**【綜合分析】** 針對特定疾病提供健康食品輔助建議。

### 參數
| 參數名 | 型別 | 必填 | 說明 | 範例 |
| :--- | :--- | :--- | :--- | :--- |
| `diagnosis_keyword` | string | 是 | 疾病名稱或 ICD-10 代碼 | `"高血壓"`, `"I10"`, `"骨質疏鬆"` |

### 用途
此工具整合了 ICD 疾病資訊與健康食品資料庫，能回答：
- 「我有糖尿病，可以吃什麼健康食品？」
- 「針對高血壓，有哪些政府核可的保健品？」

### 注意事項
回傳結果會自動包含各類必要的**醫療免責聲明**，強調健康食品非藥品，不能取代正規治療。
