# 營養工具 (Nutrition Tools)

此類別工具提供一般食品的營養成分查詢與飲食分析功能。

## search_food_nutrition
查詢食品的基礎營養資訊。

### 參數
| 參數名 | 型別 | 必填 | 說明 | 範例 |
| :--- | :--- | :--- | :--- | :--- |
| `food_name` | string | 是 | 食物名稱 | `"白米"`, `"雞胸肉"` |
| `nutrient` | string | 否 | 特定營養素過濾 | `"蛋白質"`, `"維生素C"` |

---

## get_detailed_nutrition
取得單一食品的完整營養成分表。

### 參數
| 參數名 | 型別 | 必填 | 說明 | 範例 |
| :--- | :--- | :--- | :--- | :--- |
| `food_name` | string | 是 | 精確的食物名稱 | `"糙米飯"` |

### 回傳內容
回傳所有可用的營養數據，包含熱量、三大營養素、維生素、礦物質等。

---

## search_food_ingredient
查詢食品原料法規資訊。

### 參數
| 參數名 | 型別 | 必填 | 說明 | 範例 |
| :--- | :--- | :--- | :--- | :--- |
| `keyword` | string | 是 | 原料名稱 | `"薑黃"`, `"蘆薈"` |

### 用途
確認某項原料是否可合法添加於食品中，以及其使用限制。

---

## analyze_meal_nutrition
**【飲食分析】** 計算一餐的總營養價值。

### 參數
| 參數名 | 型別 | 必填 | 說明 | 範例 |
| :--- | :--- | :--- | :--- | :--- |
| `foods` | array/list | 是 | 食物名稱列表 | `["白米", "荷包蛋", "燙青菜"]` |

### 用途
用於飲食日記分析或菜單規劃，計算整餐的熱量與營養分佈。
